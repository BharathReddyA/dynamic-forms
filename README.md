# Dynamic Forms 
### This form is useful to create the fields dynamically with specified datatype and name
### To start application in local system `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.

### API is created by Python FastAPI.
### To start the server use `python -m uvicorn main:app --reload`

 MongoDB is used as Database.
 For more package information please visit the package.json
